##clang
This repository is for test programs to make sure the gcc compiler works for Linux especially for Fedora (personal favourite). 
Should work for Microsoft environment and for all other environments also.  As long as you have the source files you can build 
your own program using your preferred gcc compiler on your OS of choice.  Building from source gives an amazing source of control 
and freedom and you always have an updated version.  Before using it as an replacement of the previous version, make sure you test 
it thoroughly before using it as an incumbant production version.  Which will then superscede the previous production's version.
